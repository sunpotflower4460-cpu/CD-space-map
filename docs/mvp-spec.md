# MVP Phase 0 仕様（原則と数学カーネル固定）

## 目的

このPhaseでは、CD星図 / Harmonic Disk の土台として原則・数学モデル・周波数変換カーネルを固定する。

## スコープ

- 原則をドキュメント化する
- 周波数を `layer / angle / rotationSpeed` に変換する関数を実装する
- 軌跡を時間の記憶として扱う前提を明記する

## 固定する考え方

- 中心は変わらない
- 1周 = 1オクターブ
- 1オクターブ上がると周波数は2倍
- 周波数比は正確に保つ
- 実際のHzは表示用にスロー化する
- 軌跡は装飾ではなく時間の記憶として扱う

## このPhaseでやらないこと

- 3D表示の作り込み
- 音声再生
- AI機能
- 複雑な物理シミュレーション
