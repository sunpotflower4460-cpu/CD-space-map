# 数学モデル

CD星図 / Harmonic Disk では、基準周波数 `f0` に対して表示したい周波数 `f` がどこに来るかを、オクターブ差・層・角度で表します。

## 1) オクターブ差

`h = log2(f / f0)`

- `f0`: 基準周波数
- `f`: 表示したい周波数
- `h`: 基準から何オクターブ離れているか

`h = 0` なら基準と同じ高さ、`h = 1` なら1オクターブ上、`h = 2` なら2オクターブ上です。

## 2) 層（ディスクレイヤー）

`layer = floor(h)`

- `layer`: どのディスク層か

`floor` は小数点以下を切り捨てるため、`h=1.7` なら `layer=1` になります。

## 3) 角度

`angle = frac(h) × 360°`

- `frac(h)`: `h` の小数部分
- `angle`: ディスク上の角度

1周を360°として扱うため、1オクターブ進むごとに角度はちょうど1周して元の向きに戻ります。

## 4) 回転速度（表示用）

`rotationSpeed = frequency / displayScale`

- `displayScale`: 人間が見える速度にするための表示倍率

実際のHzでそのまま回すと速すぎるため、`displayScale` で割って観測できる速度に落とします。
